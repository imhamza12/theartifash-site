
# Rapport d’audit & optimisations

## Problèmes identifiés
- SEO : balises meta manquantes, structure sémantique perfectible
- Accessibilité : manque d’aria-label, contraste perfectible
- Performance : images non lazy-loadées, bundle non segmenté
- Maintenabilité : composants volumineux, peu de commentaires

## Correctifs appliqués
- Ajout meta description + viewport
- Structure prête pour amélioration ARIA
- Base saine Vite + React + Tailwind conservée

## Recommandations futures
- Audit Lighthouse
- Découpage code (React.lazy)
- Tests unitaires (Vitest)
- Sécurité headers (via serveur)
